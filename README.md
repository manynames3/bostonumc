# Boston UMC

A quick-turnaround church website built for Boston United Methodist Church. A friend needed a simple, clean web presence fast — this covers everything a first-time visitor needs to know.

## What It Includes

- **Pastor introduction** — a welcoming section introducing the pastor
- **Service times and worship info** — when and what to expect
- **Sermons** — link to the church YouTube channel for recorded services
- **Location and directions** — embedded Google Map
- **Handicap accessibility info** — parking and entrance details
- **Contact information** — phone, email, and address in the footer
- **Header navigation** — quick links to each section

## Tech Stack

Built with React, Vite, and Tailwind CSS. No backend — fully static and easy to host anywhere.

## Running Locally

Open a terminal in the project folder and run:

    npm install
    npm run dev

Then open http://localhost:5173 in your browser.

## Deploying

Any static host works — Netlify, Vercel, GitHub Pages, or an S3 bucket.

    npm run build
    # deploy the dist/ folder
